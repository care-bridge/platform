# Role Based

## Donor

### Can

1. Follow hospitals to stay updated with their donation requests
2. Donate to any donation request he want
3. Donate to anynomous requests he want 
4. Donate with his preferred payment method
5. Delete his account at any time
6. Message/Contact hospitals 
7. Contact the admin
8. Get notifications with new donation requests
9. See the number of: 
    - people helped
    - donations made
    - donated money
    - last donations he made
10. Report to the admin when he get scammed by either anynomous request or hospital (hospital scam is low risk because hospitals must be verified by the admin before using the platform)

### Can't

1. Request donations
2. Delete any request on the platform
3. Can't change his role to admin/hospital
4. Can't see donations made on any request, people who donated 

## Hospital

Read this file (very important): [How to create hospital account](Accounts.md)

### Can

1. Ask for donations
2. Stop active donation goals
3. Contact the admin to change it's info/settings
4. Can see the number of donors who follow it
5. Can see the number of people donated to it's requests

### Can't

1. **Delete a request!** It can stop/finish it only
2. See donor' profiles (profiles are private for each role) 
3. Change info without contacting the admin
4. Can't delete the account at any minute, they must contact the admin
