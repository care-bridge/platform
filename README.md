# Donation Platform

Python Version: **3.11+**

## Stack

1. HTML
2. Pico.css + CSS
3. HTMX + JavaScript 
4. Python
5. Flask
6. SQLite

## Run locally

1. Clone repo
2. `python -m venv venv`
3. `source venv/bin/activate` (or `venv\Scripts\activate` on Windows)
4. Install modules: `make setup`
5. Run: `make run`
6. Open http://127.0.0.1:5000 🎉
